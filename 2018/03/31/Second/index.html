Position属性

2018年3月31日
主要了解一下css中Postion的基本使用。首先必须要知道a它的定义和用法，position属性规定元素的定位类型，任何元素都可以定位，不过绝对或固定元素会生成一个块级框（display:block)，而不论该元素本身是什么类型。相对定位元素会相对于它在正常流中的默认位置偏移。主要有以下五种:==absolute,fixed,relative,static（默认的）,inherit==.主要是了解前三种。

relative

生成相对定位的元素，相对于其正常位置进行定位。因此，”left:30” 会向元素的 left位置添加20像素。
<style type="text/css">
    h2.pos_left
    {
     position:relative;
     left:-20px
     }
     h2.pos_right
    {
     position:relative;
     left:20px
      }
</style>
<body>
<h2>处于正常</h2>
<h2 class="pos_left">向左移动</h2>
<h2 class="pos_right">向右移动</h2>>
</body>

一般使用relative来改变位置比较少，主要是用来设置子级的absolute定位的参考对象。

absolute

元素的位置通过 “left”, “top”, “right” 以及 “bottom” 属性进行规定。
效果如下
inline元素使用absolute定位之后，可以对其设置宽高；元素是不可以同时使用绝对定位和浮动的。

fixed

生成绝对定位的元素，相对于浏览器窗口进行定位。元素的位置通过 “left”, “top”, “right” 以及 “bottom” 属性进行规定。

inline元素使用absolute定位之后，可以对其设置宽高；元素是不可以同时使用fixed定位和浮动的。
特别要注意margin、padding和top，绝对定位与覆盖的问题
