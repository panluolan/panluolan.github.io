这一次介绍下鼠标点击和鼠标移出事件，实现效果弹出页面框。

使用====onmousedown==,==onmouseup==,==alert==

######

代码如下：

<div id="move"></div>  
<script type="text/javascript">  
var m = document.getElementById('move');  
var m = document.getElementById('move');
//鼠标按下
m.onmousedown = function(){//down向下
alert('鼠标点击');
}
//鼠标抬起
m.onmouseup = function(){//up向上
alert('鼠标离开');
 }
 //尺寸
var l = m.offsetLeft;//获取当前元素相对于父级元素x轴的偏移量
var t = m.offsetTop;//获取当前元素相对于父级元素y轴的偏移量
var w = m.offsetWidth;//快速获取元素的宽度
var h = m.offsetHeight;//获取元素的高度

以及css设置：

 *{margin:0px;padding:0px;}  
#move{
width:300px;
height:300px;
background:pink;
font-family:microsofyahei;
position:absolute;
top:200px;
left:400px;}

通过这个事件，可以将alert修改为其他效果，并且也可以拓展成鼠标滑过和划出事件。
